# 🇧🇷 Analyse E-commerce Olist (Brésil)
**Langue / Language :** [Français](README_FR.md) | [English](README.md) | [日本語](README_JP.md)

## 📌 Présentation du Projet
Ce projet consiste en une analyse complète (End-to-End) des données de la marketplace **Olist**. L'objectif est de démontrer ma capacité à structurer un **Data Lakehouse** sur Azure et à extraire des insights stratégiques pour la croissance de l'entreprise.

## 🏗️ Architecture des Données (Medallion)
* **Bronze :** Ingestion des données brutes au format CSV.
* **Silver :** Nettoyage, typage des données et normalisation via **PySpark**.
* **Gold :** Création d'un entrepôt de données logique avec **Serverless SQL** pour le reporting.

## 💡 Insights Stratégiques Extraits
* **Performance des Vendeurs :** Analyse de la loi de Pareto montrant une marketplace démocratique et résiliente.
* **Optimisation Logistique :** Identification des zones géographiques nécessitant des centres de distribution pour réduire les délais de livraison.
* **Analyse du Paiement :** Étude de l'impact des paiements échelonnés sur la satisfaction client (NPS).
* (Pour une analyse détaillée des recommandations stratégiques, veuillez consulter le README principal.)

## 🛠️ Compétences Démontrées
* Data Engineering avec **Azure Synapse** & **PySpark**.
* Modélisation SQL avancée (Views, Window Functions, OpenRowset).
* Business Intelligence & Data Storytelling.
